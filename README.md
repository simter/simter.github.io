# [Simter 网站源码](https://github.com/simter/simter.github.io/tree/source)

访问 [Simter 网站](http://simter.github.io)。

## 目录结构

```
src                             - 网站源码
dist                            - src 源码的打包发布目录
test                            - 测试
```

## 构建

```bash
npm run build
```

## Demo

```bash
npm start
```

然后用浏览器访问 <http://127.0.0.1:3000>。
