# [Simter 文档](http://simter.tech)

## 目录结构

```
index.html                      - 网站入口文件
data                            - 网站使用的 json 数据文件
libs                            - 第三方依赖库

src                             - 源代码
test                            - 测试
dist/index.js                   - src 下各组件用 Rollup 打包后的文件
```

## 构建

```bash
npm build
```
